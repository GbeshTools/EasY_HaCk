instalation of EasY_HaCk tool :
1) pkg install update && upgrade
2) pkg install git 
3) git clone https://github.com/sabri-zaki/EasY_HaCk 
4) cd EasY_HaCk/
5) chmod +x setup
6) ./setup
7) ./EasY_HaCk.sh
8) enjoy
